\# Matrimony Event Booking System



A full-stack web application for managing and booking matrimony and professional events.



\## Tech Stack

\- Frontend: Angular

\- Backend: Node.js, Express, TypeScript

\- Database: MySQL



\## Features

\- View matrimony and professional events

\- Book tickets for events

\- Backend validation for seat availability

\- REST API integration

\- MySQL database connectivity
